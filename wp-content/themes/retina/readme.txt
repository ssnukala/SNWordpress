== Description ==

Description: An elegant, versatile, nice and simple blog theme featuring custom background, custom header, custom menu, flexible header, featured images, sticky post, fixed-width, theme options, widget-ready, threaded comments and translation ready. You can easily use your own Logo, Text and Background. The Theme can work for various different niches.

== Note ==

"Custom Header" suggested width is 1920 pixels. Suggested height is 400 pixels. Retina is using "background-size: cover;" property to fill the entire width so make sure to use the image of suggested dimensions for optimum results.
"Site Title" will work best for five to six words.
"Site Description" will work best for twelve to fifteeen words.
"Primary Menu" will work best for six to eight menu items.
See the "Help" tab in the "Retina Options" admin page for license, changelog and other informations.

== Installation ==

Manual installation:

1. Upload the `retina` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'retina'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Retina Options for Theme Options
